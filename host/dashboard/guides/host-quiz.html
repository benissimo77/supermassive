Notes for Hosts

Hosting a quiz is easy! Follow these simple steps to get started:

Now let's look at those steps in more detail...

  1. Host a Game Room

    Start by selecting a quiz from the Quiz Dashboard. You can start a brand new quiz or select from the list of available quizzes. Selecting a quiz takes you to the Quiz Editor where you can adjust the quiz as much as you like. See the separate guide for details on how to use the quiz editor.

    After reviewing your quiz in the editor, you can save the quiz for later or start hosting immediately. When you are ready to host, click the "Host Quiz" button at the top of the Quiz Editor. You will then be LIVE and ready for teams to join!

  2. Invite your Friends

    The most common set-up is that everybody is in the same physical space and your laptop is connected to a large central screen or projector so that everyone can see the questions and answers clearly. The initial quiz 'lobby' display shows a QR code for teams to scan and join.

    If you are live-streaming your quiz to a remote audience teams can still join in the same way. But be aware you need to allow for some delay due to streaming latency. We have a separate Twitch streaming guide especially for this use-case.

    If teams can't join via the QR code then they can manually enter via videoswipe.net. Instructions for this are also shown in the quiz lobby display.

    Each time a new team joins they are shown on the host screen, so everyone can see who is currently about to play.

    NOTE: we have talked about 'teams' here, but most often a quiz is played in teams. The key point is that each team uses only one device to answer their questions, so a team and a team are effectively the same.

  3. Everyone watches the host screen

    So... the teams have all joined and you are ready to actually start the quiz! As host you control the flow of the quiz: use the arrow keys LEFT and RIGHT to step forward/backward through the quiz one question at a time.

    If teams want to repeat a question you can use the LEFT arrow key to step backward through the questions. Use SHIFT + arrow keys to step to the next or previous round.

  4. Teams use their devices to answer questions

    After each question is displayed, teams can submit their answers via their device. Only one device is used per team. The host screen shows whenever a team has submitted their answer - as host you can see once ALL teams have submitted an answer, or call out the teams who haven't answered and hurry them along.

    Optionally, you can elect to use a countdown timer for questions, and teams have a limited amount of time to submit their answers. This option can be selected in the Quiz Editor. Each round has its own timer setting. If you're using a timer the host screen will show a countdown clock for each timed question.

That's it! You are now ready to host your own quiz. Here are a few pro tips that we've found can be useful when hosting:

  • Make sure you have a good internet connection - the host screen and team devices need to be able to communicate with the server in real-time.
  • It's worth 'rehearsing' your quiz before hosting it for a large group. This will help you identify any issues and ensure that everything runs smoothly. We usually host the quiz and join using a phone to test the host and player experience at the same time.
  • For a high-impact dramatic ending you could try including a final 'quick-fire' round which shows answers and updates scores after every question. These settings are available from the Quiz Editor.

Good luck and have fun!
