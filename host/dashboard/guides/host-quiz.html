Notes for Hosts

Hosting a quiz is easy! Follow these simple steps to get started:

  1. Select a Quiz: Go to the Quiz Dashboard and choose the quiz you want to host.
  2. Start the Game: Click the "Host Quiz" button. This will generate a unique room code.
  3. Invite Players: Share the room code with your players. They can join by entering the code on the play page.
  4. Control the Flow: As the host, you control when to move to the next question and when to show the leaderboard.
