Using the Quiz Editor

The Quiz Editor allows you to create and customize your own quizzes.

  • Rounds: Organize your quiz into rounds. You can add, delete, and reorder rounds.
  • Questions: Add different types of questions to each round, including Multiple Choice, True/False, and more.
  • Images: Enhance your questions by adding images from the Image Library.
  • Saving: Don't forget to save your changes! The save button will highlight when there are unsaved changes.
