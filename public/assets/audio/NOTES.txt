A few thoughts on the audio files in this folder so I don't end up playing them all 15 times before deciding which ones to use...

There are two types of music for the quiz:
 - intro and theme music (between rounds, end, beginning etc)
 - countdown music (10, 30, 60 seconds timer)

This folder contains mainly the first kind as this has the most variety
I found quite a few funky type ones which could work well, but it needs to be considered along with the overall vibe of the quiz
Maybe I want to create several 'themes' (eg Corporate, Party, Classic, Childrens, Christmas...) which would determine not only the audio
but also the colours and visual design of the quiz.
So maybe I should decide one say three themes and just run with those.
OR make 1 (ONE) theme only and not worry about themes for now!
I'm thinking probably the funky comedy quiz would work best because that plays to my strengths, but that means pitching the whole thing
as a comedy game right off the bat which might affect things like animations - eg cartoon style

them all.
